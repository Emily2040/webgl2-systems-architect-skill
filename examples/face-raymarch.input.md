# Example Input - Face Raymarch

I want a WebGL 2.0 face-bust renderer for mobile Safari and desktop Chrome.
The first target is a phone-sized canvas at 60 FPS.
It should look anatomically credible from medium-close framing, not just like a smooth mannequin.
I expect SDF raymarching, soft shadows, AO, and some lip/eye material separation.
I care about a stable startup, clear feature priorities, and a path for lower-end devices.
Please design the renderer and note what can run in parallel or asynchronously.
